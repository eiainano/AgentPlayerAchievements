# Pixel Art Ideas — AGPA 成就像素画描述 (A-Z 排序)

> 为每个 AGPA 成就设计的像素画概念描述。描述需与成就中英文名、中英文说明一致，可适当联想扩展，但不过分。
> 像素画风格参考：限制调色板，明确前景/背景构成，64×64 或 32×32 点阵感。
> 所有生图统一使用 **1:1 正方形比例**。
> 
> 本文件是 pixel-art-ideas.md 的全局 A-Z 排序版本，所有成就按 ID 字母序排列，不分类别。
