### 展示柜管理

```bash
/achievements showcase <id1> <id2> ... <id6>  # 设置展示柜
/achievements showcase --auto                   # 自动选最稀有的 6 个
```

## Agent Level 系统

成就 = 离散里程碑，等级 = 连续增长感。两者互补。

### EXP 来源

```
+ 完成 task             10 XP
+ task 复杂性 bonus      跨文件 +5, tool chain 长 +5
+ 使用新功能             20 XP（仅首次）
+ 解锁成就              50-500 XP（取决于稀有度）
+ 连续使用 bonus         streak day N → +N XP
+ 分享成就               10 XP
```

### 等级公式

```
Level = floor(sqrt(Total_XP / 100))

Level 1    = 100 XP
Level 10   = 10,000 XP
Level 50   = 250,000 XP
Level 100  = 1,000,000 XP
```

### 等级特权（纯装饰）

```
Lv 10:  解锁自定义展示柜背景色
Lv 25:  展示柜从 6 格升级到 9 格
Lv 50:  解锁金色 profile 边框
Lv 100: 自定义成就名称颜色
```

## 进度与反馈 UI

### 解锁瞬间（终端效果）

agent 输出完毕，用户输入前，在 prompt 上方插入：

```
─────────────────────────────────────
  🏆 Achievement Unlocked!
  "The Debugger"
  agent 代码报错，你让它自己修，
  它修好了。这就是信任。
  Rarity: Uncommon · Top 45%
  Set: The Bug Catcher (1/2)
─────────────────────────────────────
> _   ← 用户输入光标
```

macOS 系统通知作为双保险（用户切窗口时也能看到）：

```
osascript -e 'display notification "agent 代码报错，它自己修好了"
  with title "🏆 Achievement Unlocked: The Debugger"
  subtitle "Uncommon · Top 45%"'
```

## CLI 命令接口

```bash
# 成就
/achievements                  # 所有成就（已解锁 / 未解锁 / 进度）
/achievements recent           # 最近解锁的 5 个
/achievements next             # 最接近解锁的 5 个（按进度排序）
/achievements show <id>        # 单个成就详情
/achievements showcase <ids>   # 设置展示柜
/achievements export           # 导出成就数据 JSON
/achievements import <file>    # 导入（换机器/迁移）

# 个人资料
/profile                       # 资料页（等级、成就、统计）
/profile share                 # 生成分享用成就卡片

# 系统
/achievements stats            # 累计统计
/achievements history          # 解锁历史时间线
```
